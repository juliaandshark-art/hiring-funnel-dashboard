# Hiring Funnel Dashboard

Интерактивный HTML-дашборд эффективности воронки найма Л'Этуаль по срезу Атея от 03.07.2026.

## Что внутри

- KPI по кандидатской воронке: лиды, контакт, приглашение, назначенное интервью, show-up, стажировка, оформление.
- Фильтры по дате, региону, городу, роли, рекрутеру, источнику, типу отклика и статусам.
- SLA-слайдеры по срокам до интервью, оформления и закрытия вакансии.
- Блоки по источникам, рекрутерам, причинам потерь, вакансиям и проблемным зонам.

## Откуда берутся цифры

Данные заранее рассчитаны из двух Excel-исходников от 03.07.2026 и встроены внутрь `index.html`. Страница не обращается к Excel-файлам после публикации и работает как автономный статический дашборд.

## Как пользоваться

1. Откройте опубликованную GitHub Pages-ссылку.
2. Используйте фильтры в верхней части страницы, чтобы сузить срез по региону, городу, роли, рекрутеру, источнику или статусу.
3. Меняйте SLA-слайдеры, чтобы смотреть долю этапов, уложившихся в целевой срок.
4. KPI, графики и таблицы пересчитываются автоматически после изменения фильтров.

## Как обновлять данные

Для обновления нужно заново собрать `index.html` из новых Excel-файлов и заменить файл в репозитории. Текущая версия не подгружает новые Excel напрямую на сайте.

## Важно

Репозиторий публичный. В опубликованной версии не размещены исходные Excel-файлы и удалены локальные пути к файлам. Внутри HTML остаются данные, необходимые для работы дашборда.
